# JPA
- VO(Value Object) 클래스와 테이블 관계
```
사용자가 입력한 데이터나 비즈니스 로직 수행 결과로 얻은 데이터를 재사용할 수 있도록 데이터베이스에 저장한다.

자바 객체와 데이터베이스의 테이블 사이를 매핑하기 위해 많은 SQL 구문과 자바 코드가 필요할 수 밖에 없다.
```
