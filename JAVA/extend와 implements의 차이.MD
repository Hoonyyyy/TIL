# 차이

## extends
- 일반클래스와 abstract 클래스 상속
- 클래스 한 개만 상속 받을 수 있다.
- 자신 클래스는 부모 클래스의 기능을 사용한다.

## implement
- interface 상속에 사용
- interface가 class를 사용할 떈 implements를 쓸 수 없다.
- 여러개 사용 가능
- 설계 목적으로 구현 가능하다
- implement한 클래스는 implements의 내용을 다 사용해야 한다


일반 클래스와 추상 클래스를 상속할때 extends를 사용하고

interface를 상속할때는 implements를 사용한다.
