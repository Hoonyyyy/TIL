# for문 향상for문 while문 차이


### 여기는 정리 다시하기
- for : 반복횟수가 정해져 있다.
- 향상for문 : 순차적으로만 가능 ( 단순 입력용 )
- while : 무한루프일 경우에 쓴다( 반복 조건을 검사하여 false면 시작조차 하지않음)
- do-while : 먼저 한번은 반복코드 실행 후, 반복 조을 검사하여 조건이 false면 반복을 멈춤
